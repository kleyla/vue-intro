# INTRO TO VUE JS

https://cli.vuejs.org/

Create new project

```
vue create my-project
```

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
